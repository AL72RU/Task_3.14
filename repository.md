## Создание репозитория

---

Для создания репозитория на GitHub нужно:
1. Войти в систему

2. Перейти в раздел "**Repositories**"

3. Нажать на зелёную кнопку "**New**"

4. Заполнить обязательные поля:
 
    * ***Repository name***
    * ***Public*** или ***Private***
    
5. Нажать кнопку "**Create repository**"
